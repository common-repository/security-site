== Feature == 
Support detect XSS Attack your site: 
 
* Search  
* Form 
* Contact form 7 
* Sendmail 
* PHP mailer

#### Thank you for your interest in `Security Site`.
### Minimum requirements. 

* Wordpress 4.0+
* PHP 5.x
* MySQL 5.x
 
This section describes how to install the plugin and get it working. 
e.g.  

1. Upload folder 'plugin-name' to the `/wp-content/plugins/` directory.
2. Activate the "Security Site" plugin through the plugins menu in WordPress.
